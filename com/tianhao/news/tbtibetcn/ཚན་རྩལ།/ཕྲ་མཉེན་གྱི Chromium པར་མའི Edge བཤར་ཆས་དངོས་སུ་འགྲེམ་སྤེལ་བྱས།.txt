ཕྲ་མཉེན་གྱི Chromium པར་མའི Edge བཤར་ཆས་དངོས་སུ་འགྲེམ་སྤེལ་བྱས།
དེ་ཡང་སྤྱི་ལོ 2018 ལོའི་ཟླ 12 པར། ཕྲ་མཉེན་གྱིས Chromium རྨང་གཞི་བྱས་པའི་ཐོན་རིམ་གསར་བའི Edge བཤར་ཆས་གསར་སྤེལ་བྱ་རྒྱུ་ཡིན་པའི་སྐོར་བསྒྲགས་མྱོང་བ་དང་། ཕྲ་མཉེན་གྱི་དམིགས་འབེན་ནི་སྤྱོད་མཁན་ལ་སྔར་བས་ལེགས་པའི Web འཆམ་མཐུན་རང་བཞིན་གསར་བཟོ་བྱེད་པ་དེ་ཡིན་ལ། དེ་དང་ལྷན་དུ Web གསར་འབྱེད་མི་སྣར Web ཧྲུག་འགྱུར་ཇེ་ཉུང་དུ་གཏོང་བ་དེ་རེད། ལོ་ངོ་གཅིག་གི་རིང་ལ་ཐོན་རིམ་གསར་བའི Edge བཤར་ཆས་ཚོད་ལྟ་བྱས་མཐར། ཕྲ་མཉེན་གྱིས་དེ་རིང་དངོས་སུ Windows དང macOS ལ་འཚམ་པའི Microsoft Edge བཤར་ཆས་ཕྱོགས་ཡོངས་ནས་ཚོང་རར་འདོན་པ་ཁྱབ་བསྒྲགས་བྱས།
          ཁྱེད་རང་གིས Microsoft Edge དྲ་ཚིགས་(https://www.microsoft.com/en-us/edge) བརྒྱུད་ནས་སྐད་ཡིག 90 ལྷག་ལ་རྒྱབ་སྐྱོར་ཐོབ་པའི Windows ཡི Microsoft Edge བཤར་ཆས་ཀྱི་“བརྟན་ལྷིང་”གི་པར་གཞི་ངོ་མ་ (Microsoft Edge 79 བརྟན་ལྷིང་པར་གཞི) ཕབ་ལེན་བྱེད་ཆོག་པ་དང་། དུས་མཚུངས་སུ Windows 8.1/8, Windows 7, macOS, iOS དང་ཨན་གྲོལ་མ་ལག་བཅས་སོགས་ལའང་དུས་མཉམ་དུ་རྒྱབ་སྐྱོར་ཐོབ་པ་ཡིན།
          གལ་སྲིད་ཁྱེད་རང་གིས Microsoft Edge བཤར་ཆས་དེ་ལག་སྒུལ་སྒྲིག་འཇུག་བྱེད་མི་འདོད་ཚེ། མ་འོངས་པའི་དུས་སུ་དེའི Windows 10 ཡི་གསར་བཅོས་རང་འགུལ་སྒྲིག་འཇུག་ལ་སྒུག་ཀྱང་ཆོག་པ་ཡིན། ཕྲ་མཉེན་གྱིས་དེའི་རྗེས་ཀྱི་ཟླ་འགའི་ནང Windows 10 སྤྱོད་མཁན་ཡོངས་ལ་གཏན་འཇགས་ཀྱི་པར་གཞི་གསར་པ་འདོན་རྒྱུ་རེད། གནས་སྤོའི་བརྒྱུད་རིམ་དེ་རྗེས་ཀྱི་གཟའ་འཁོར་འགའ་ནས་འགོ་ཚུགས་རྒྱུ་རེད།
